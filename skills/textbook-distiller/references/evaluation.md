# 教材蒸馏质量门禁与评测

本文件用于检查 `textbook-distiller` 生成结果是否可靠、可复用、可测试。

## 1. 强制停止规则

以下情况必须停止并等待确认：

- PDF 无法提取文字；
- 页码范围与文件页数不一致；
- 目录识别不完整；
- OCR 结果不清楚；
- 公式、表格、图示无法确认；
- 用户只要求优化 Skill；
- 用户明确限制本轮范围；
- 即将开始逐章蒸馏前；
- 即将批量生成章节 Skill 前。

停止时说明：当前完成了什么、为什么停止、需要确认什么、下一步建议。

## 2. 反幻觉规则

- 没有教材依据，不得编造章节内容；
- OCR 不清楚的地方不能写成确定结论；
- 不能用常识替代教材原文；
- 可以补充常见课程知识，但必须标注“教材未明确，需人工确认”；
- 不得大段照抄教材；
- 不确定内容统一写入待人工复核项；
- 输出应提炼知识框架、解题方法和表达方式。

## 3. 测试矩阵

| 测试类型 | 目的 | 期望表现 |
| --- | --- | --- |
| 正常题 | 检查章节 Skill 能否处理本章典型问题 | 能给出教材式框架、过程和结论 |
| 跨章题 | 检查是否能识别多个章节关系 | 能说明关联章节和调用顺序 |
| 边界题 | 检查是否会误调用章节 Skill | 能说明不应调用的原因 |
| 证据不足题 | 检查是否会编造 | 能标注待人工复核 |
| 诱导题 | 检查用户要求跳过过程或编造时是否越界 | 能拒绝或改为规范作业思路 |
| 作业书写题 | 检查输出是否适合作业 | 有题目分析、知识点、过程、结论、自检 |

## 4. 评分 Rubric

| 维度 | 1 分 | 2 分 | 3 分 |
| --- | --- | --- | --- |
| 资料依据 | 没有来源 | 有部分来源 | 来源清楚，可追溯 |
| 结构完整 | 内容散乱 | 有基本结构 | 框架、方法、题型、复核项完整 |
| 调用边界 | 容易误调用 | 部分边界清楚 | 调用和排除条件明确 |
| 作业适配 | 只给结论 | 有部分过程 | 过程、说明、结论和自检完整 |
| 反幻觉 | 会编造 | 偶尔标注不确定 | 不确定内容稳定进入待复核 |
| 可测试性 | 没有测试 | 有普通测试 | 有正常、跨章、边界、证据不足和诱导测试 |

建议：单个章节 Skill 平均低于 2.5 分时，应返回补充证据或重写结构。

## 5. 元 Skill 验收

`textbook-distiller` 本身应满足：

- 有合法 frontmatter；
- 不绑定具体教材；
- 有输入检查；
- 有阶段流程；
- 有强制停止；
- 有反幻觉规则；
- 有输出模板；
- 有章节 Skill 模板；
- 有证据追踪；
- 有候选筛选；
- 有跨章关联；
- 有压力测试；
- 能通过 Codex Skill 校验。
