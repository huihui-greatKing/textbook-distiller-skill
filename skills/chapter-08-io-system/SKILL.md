## 章节知识框架

- I/O 接口：连接主机和外设，完成数据缓冲、状态控制和信号转换。
- 端口：CPU 与 I/O 设备交换信息的寄存器或地址。
- 程序查询：CPU 主动轮询设备状态，简单但效率低。
- 中断：设备主动请求 CPU 服务，提高 CPU 利用率。
- DMA：外设与主存直接传输数据，减少 CPU 参与。
- 总线：连接 CPU、主存和外设的数据、地址、控制通道。

## 解题步骤

1. 先判断题目是否属于第8章；
2. 提取题目关键词和已知条件；
3. 对应本章核心概念或规则；
4. 按教材式步骤展开分析或计算；
5. 写出结论，并说明适用条件；
6. 对 OCR 不清楚的公式、图表或例题标注待复核。

## 教材式表达模板

- 概念解释：本概念用于……，它解决的问题是……，在本章中通常和……一起使用。
- 比较分析：二者的共同点是……，区别主要体现在……，适用场景分别是……。
- 计算/过程题：已知……，根据……规则，先……，再……，最后得到……。
- 作业总结：因此，本题的关键不是只写结果，而是写清判断依据和过程。

## 常见错误

- 只写结论，不写判断依据；
- 把相邻章节概念混用；
- 使用公式但不写适用条件；
- 忽略 OCR 中公式、表格和图示可能识别错误；
- 作业答案缺少分点和过程。

- 是否适合直接改写成作业答案；
- 是否标注待复核公式、图表或 OCR 内容；
