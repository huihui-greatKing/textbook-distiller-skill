# 综合作业 Skill

## Skill 目标

本 Skill 用于把课程作业题写成可提交前参考的完整答案草稿。它不是只告诉用户“用什么方法”，而是要完成章节判断、题型判断、知识点定位、详细过程、最终答案、易错点和检查方法。

## 调用流程

1. 读取题目，提取关键词、已知条件和问题要求；
2. 判断所属章节，可以是单章或跨章；
3. 判断题型：概念辨析、简答、计算、分析、比较、综合、实验或设计；
4. 调用对应章节 Skill；
5. 按题型生成完整作业式答案；
6. 对 OCR、公式、图表或教材例题不确定处标注“需人工复核”。

## 通用输出格式

### 题目分析

说明题目问什么、给了什么条件、隐藏考点是什么。

### 所属章节与知识点

列出章节、章节 Skill、核心概念和需要调用的规则。

### 解题思路

说明为什么选择这个方法，先处理什么，再处理什么。

### 详细解题过程

按教材式步骤展开，计算题保留关键计算，简答题保留分点说明。

### 最终答案

给出可以写入作业的结论。

### 易错点

列出容易混淆、漏写或算错的地方。

### 检查方法

说明如何检查章节、公式、单位、结果范围和表达完整性。

## 章节路由提示

- 第1章：计算机系统、组成、性能、存储程序思想；
- 第2章：数据表示、机器数、补码、浮点数、校验码；
- 第3章：运算方法、补码运算、溢出、浮点运算、ALU；
- 第4章：指令格式、操作码、地址码、寻址方式、汇编；
- 第5章：CPU、数据通路、控制器、指令周期、微程序；
- 第6章：流水线、吞吐率、加速比、结构/数据/控制相关；
- 第7章：RAM、ROM、Cache、存储层次、虚拟存储；
- 第8章：I/O 接口、中断、DMA、总线、外设传送；
- 第9章：多机系统、并行处理、多处理机、多计算机、互连网络；
- 第10章：EDA、HDL、仿真、综合、验证、设计报告。

## 不应输出的内容

- 只写“本题可用某方法”；
- 只给最终答案，不写过程；
- 编造教材没有依据的公式、图号或例题；
- 把 OCR 不确定内容写成确定结论。
