# 公式与表格复核记录

OCR 对公式、上下标、特殊符号和表格列边界不稳定。本文件记录需要重点复核的内容类型。

| 章节 | 复核对象 | 复核原因 | 处理建议 |
| --- | --- | --- | --- |
| 第1章 | 性能指标公式和单位 | OCR 容易混淆符号和单位 | 对照原 PDF 修正 |
| 第2章 | 数制转换、补码、浮点数和校验码公式 | 上下标、负号、指数容易识别错误 | 人工确认后再用于计算题 |
| 第3章 | 加减乘除运算过程和溢出判断 | 运算符和进位位易错 | 保留教材步骤并人工复核 |
| 第4章 | 指令格式表、寻址方式表 | 表格列边界可能丢失 | 对照原图表整理 |
| 第5章 | 控制信号表、微指令字段 | 表格和缩写较多 | 逐项校对字段含义 |
| 第6章 | 吞吐率、加速比、效率等公式 | 下标、分母和周期数易错 | 计算题前复核 |
| 第7章 | Cache 命中率、存储带宽、地址划分 | 公式和地址字段容易被 OCR 拆散 | 按原 PDF 重建表格 |
| 第8章 | I/O 传输率、中断/DMA 时序 | 单位和时序字段易错 | 与图表一起复核 |
| 第9章 | 并行性能和加速比公式 | 符号、参数含义需确认 | 标注参数解释 |
| 第10章 | HDL 代码、时序表达式 | 代码字符易错 | 不直接复制 OCR 代码 |
