# 第9章 多机系统 知识点提炼

| 知识点 | 解释 | 章节 | 用途 |
| --- | --- | --- | --- |
| 多机系统 | 由多个处理单元协同完成任务的系统。 | 第9章 | 复习、作业、章节 Skill 调用 |
| 并行处理 | 多个操作同时或重叠执行以提高处理能力。 | 第9章 | 复习、作业、章节 Skill 调用 |
| 多处理机 | 多个处理器通常共享存储资源。 | 第9章 | 复习、作业、章节 Skill 调用 |
| 多计算机 | 多个计算机节点通过网络互连，常通过消息传递协作。 | 第9章 | 复习、作业、章节 Skill 调用 |
| 互连网络 | 连接处理器、存储器或节点的通信结构。 | 第9章 | 复习、作业、章节 Skill 调用 |
| 加速比 | 并行系统相对串行系统的性能提升程度。 | 第9章 | 复习、作业、章节 Skill 调用 |

## 复习建议

多机系统题不要只写“多个 CPU”，要说明存储组织、通信方式和并行开销。
