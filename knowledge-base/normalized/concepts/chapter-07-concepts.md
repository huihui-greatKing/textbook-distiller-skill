# 第7章 存储系统 知识点提炼

| 知识点 | 解释 | 章节 | 用途 |
| --- | --- | --- | --- |
| 存储层次 | 由寄存器、Cache、主存、外存等组成，越靠近 CPU 通常越快、容量越小、成本越高。 | 第7章 | 复习、作业、章节 Skill 调用 |
| RAM | 随机读写存储器，主存常用类型。 | 第7章 | 复习、作业、章节 Skill 调用 |
| SRAM | 静态 RAM，速度快、成本高，常用于 Cache。 | 第7章 | 复习、作业、章节 Skill 调用 |
| DRAM | 动态 RAM，容量大、成本低，需要刷新，常用于主存。 | 第7章 | 复习、作业、章节 Skill 调用 |
| ROM | 只读或以读为主的非易失存储器。 | 第7章 | 复习、作业、章节 Skill 调用 |
| Cache | 位于 CPU 和主存之间，用局部性原理提高平均访存速度。 | 第7章 | 复习、作业、章节 Skill 调用 |
| 命中率 | CPU 访问数据在 Cache 中找到的比例。 | 第7章 | 复习、作业、章节 Skill 调用 |
| 替换算法 | Cache 满时选择被替换块的方法。 | 第7章 | 复习、作业、章节 Skill 调用 |
| 虚拟存储器 | 通过主存和外存配合，让程序看到更大的地址空间。 | 第7章 | 复习、作业、章节 Skill 调用 |
| 页表 | 记录虚拟页和物理页框对应关系的数据结构。 | 第7章 | 复习、作业、章节 Skill 调用 |

## 复习建议

存储系统题要把“为什么这样设计”和“怎么算”结合起来，尤其 Cache 题要写映射和替换依据。
