# 第2章 计算机中的数据表示 知识点提炼

| 知识点 | 解释 | 章节 | 用途 |
| --- | --- | --- | --- |
| 机器数 | 计算机内部实际存放的二进制编码形式。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 真值 | 机器数所代表的实际数值，是人理解的数学值。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 原码 | 最高位表示符号，其余位表示绝对值，直观但加减运算不方便。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 补码 | 负数通过模运算思想表示，可把减法转化为加法，是计算机常用的有符号数表示。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 反码 | 负数除符号位外各位取反，常作为理解补码的过渡。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 定点数 | 小数点位置固定，适合整数或固定小数范围的表示。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 浮点数 | 用阶码和尾数表示数值范围较大的数，答题时要分符号、阶码、尾数说明。 | 第2章 | 复习、作业、章节 Skill 调用 |
| BCD 码 | 用二进制编码十进制数，适合十进制输入输出场景。 | 第2章 | 复习、作业、章节 Skill 调用 |
| ASCII/汉字编码 | 用于表示字符或汉字，属于非数值数据编码。 | 第2章 | 复习、作业、章节 Skill 调用 |
| 奇偶校验与 CRC | 用于检错，答题时要说明校验位或生成多项式的作用。 | 第2章 | 复习、作业、章节 Skill 调用 |

## 复习建议

数据表示题最容易丢步骤。作业中应写清位数、符号位、转换过程、最终编码和是否需要复核公式符号。
