# 第5章 处理器(CPU)设计 知识点提炼

| 知识点 | 解释 | 章节 | 用途 |
| --- | --- | --- | --- |
| CPU | 由运算器、控制器、寄存器等组成，负责取指、译码和执行。 | 第5章 | 复习、作业、章节 Skill 调用 |
| 数据通路 | 数据在寄存器、ALU、存储器和总线之间流动的路径。 | 第5章 | 复习、作业、章节 Skill 调用 |
| 控制信号 | 控制各部件在正确时序完成操作的信号。 | 第5章 | 复习、作业、章节 Skill 调用 |
| 硬布线控制 | 用组合逻辑和时序逻辑产生控制信号，速度快但修改困难。 | 第5章 | 复习、作业、章节 Skill 调用 |
| 微程序控制 | 用微指令组织控制过程，结构清晰且便于修改。 | 第5章 | 复习、作业、章节 Skill 调用 |
| 指令周期 | 一条指令从取指到执行完成所经历的过程。 | 第5章 | 复习、作业、章节 Skill 调用 |

## 复习建议

CPU 题要把“数据通路”和“控制信号”一起写，否则答案容易空泛。
