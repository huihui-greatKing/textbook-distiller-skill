# 第6章 流水线技术 知识点提炼

| 知识点 | 解释 | 章节 | 用途 |
| --- | --- | --- | --- |
| 流水线 | 把指令执行划分为多个阶段，使多条指令在不同阶段重叠执行。 | 第6章 | 复习、作业、章节 Skill 调用 |
| 吞吐率 | 单位时间内完成的任务或指令数量。 | 第6章 | 复习、作业、章节 Skill 调用 |
| 加速比 | 非流水执行时间与流水执行时间之比。 | 第6章 | 复习、作业、章节 Skill 调用 |
| 效率 | 流水线各段实际利用程度。 | 第6章 | 复习、作业、章节 Skill 调用 |
| 结构相关 | 多条指令争用同一硬件资源产生的冲突。 | 第6章 | 复习、作业、章节 Skill 调用 |
| 数据相关 | 后一条指令需要前一条指令结果导致的等待。 | 第6章 | 复习、作业、章节 Skill 调用 |
| 控制相关 | 分支转移导致下一条指令地址不确定。 | 第6章 | 复习、作业、章节 Skill 调用 |

## 复习建议

流水线题要画清阶段和周期，公式必须结合题目给定段数和任务数。
