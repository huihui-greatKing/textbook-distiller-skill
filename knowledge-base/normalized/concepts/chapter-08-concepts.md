# 第8章 输入/输出系统 知识点提炼

| 知识点 | 解释 | 章节 | 用途 |
| --- | --- | --- | --- |
| I/O 接口 | 连接主机和外设，完成数据缓冲、状态控制和信号转换。 | 第8章 | 复习、作业、章节 Skill 调用 |
| 端口 | CPU 与 I/O 设备交换信息的寄存器或地址。 | 第8章 | 复习、作业、章节 Skill 调用 |
| 程序查询 | CPU 主动轮询设备状态，简单但效率低。 | 第8章 | 复习、作业、章节 Skill 调用 |
| 中断 | 设备主动请求 CPU 服务，提高 CPU 利用率。 | 第8章 | 复习、作业、章节 Skill 调用 |
| DMA | 外设与主存直接传输数据，减少 CPU 参与。 | 第8章 | 复习、作业、章节 Skill 调用 |
| 总线 | 连接 CPU、主存和外设的数据、地址、控制通道。 | 第8章 | 复习、作业、章节 Skill 调用 |

## 复习建议

I/O 题要围绕“谁控制传输、CPU 是否等待、效率如何”来组织答案。
