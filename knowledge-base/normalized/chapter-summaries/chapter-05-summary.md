# 第5章 处理器 CPU 设计 学习型蒸馏总结

## 本章不是概要，而是学习任务

理解 CPU 如何通过数据通路和控制器完成取指、译码和执行。

## 学习目标

- 说明 CPU 基本组成
- 描述指令周期
- 分析数据通路
- 比较硬布线控制和微程序控制

## 知识地图

- CPU：运算器、控制器、寄存器
- 指令执行：取指、译码、执行、访存、写回
- 控制器：硬布线控制、微程序控制

## 概念学习法

学习本章概念时，不只背定义，而要按“是什么、为什么引入、怎么用、怎么考、易混点”整理。重点概念包括：CPU、数据通路、控制信号、硬布线控制、微程序控制。

## 作业解题法

本章作业应先定位题型，再选择规则或方法。常用方法包括：指令执行五阶段、控制器比较。答案要保留题目分析、已知条件、过程、结论和检查。
