# 第5章 处理器(CPU)设计 题型与作业方法

| 题型 | 常见考法 | 解题/作业写法 |
| --- | --- | --- |
| 指令执行流程题 | 说明一条指令如何在 CPU 中执行。 | 按阶段列出寄存器传送和控制信号。 |
| 数据通路分析题 | 分析部件连接和数据流。 | 结合图写出数据流向。 |
| 控制方式比较题 | 比较硬布线与微程序。 | 从实现方式、速度、可修改性比较。 |

## 通用答题模板

1. 判断题目涉及的知识点；
2. 写出教材中的相关概念、规则或公式；
3. 按步骤展开分析或计算；
4. 给出结论；
5. 标注需要人工复核的 OCR、公式或图表内容。
