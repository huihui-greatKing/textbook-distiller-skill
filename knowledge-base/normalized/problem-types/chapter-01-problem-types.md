# 第1章 绪论 题型与作业方法

| 题型 | 常见考法 | 解题/作业写法 |
| --- | --- | --- |
| 概念解释题 | 解释计算机系统、硬件、软件、指令、程序等基本概念。 | 先给定义，再说明作用，最后联系后续章节。 |
| 组成结构题 | 说明计算机由哪些基本部件组成。 | 按五大部件写，并说明各部件之间通过数据和控制信息配合。 |
| 性能指标题 | 比较主频、字长、吞吐率、响应时间等指标。 | 不要只说越大越好，要说明指标适用场景。 |

## 通用答题模板

1. 判断题目涉及的知识点；
2. 写出教材中的相关概念、规则或公式；
3. 按步骤展开分析或计算；
4. 给出结论；
5. 标注需要人工复核的 OCR、公式或图表内容。
