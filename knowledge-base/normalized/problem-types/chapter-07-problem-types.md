# 第7章 存储系统 题型与作业方法

| 题型 | 常见考法 | 解题/作业写法 |
| --- | --- | --- |
| 存储器分类题 | 比较 RAM/ROM/SRAM/DRAM。 | 从是否易失、速度、成本、用途比较。 |
| Cache 分析题 | 求命中率、平均访存时间或映射关系。 | 先判断映射方式，再分析标记、索引和块内地址。 |
| 虚拟存储题 | 地址转换或页表分析。 | 拆分虚拟地址，查页表，再得到物理地址。 |

## 通用答题模板

1. 判断题目涉及的知识点；
2. 写出教材中的相关概念、规则或公式；
3. 按步骤展开分析或计算；
4. 给出结论；
5. 标注需要人工复核的 OCR、公式或图表内容。
