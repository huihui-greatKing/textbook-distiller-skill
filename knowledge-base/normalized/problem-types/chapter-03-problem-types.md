# 第3章 运算方法与运算器 题型与作业方法

| 题型 | 常见考法 | 解题/作业写法 |
| --- | --- | --- |
| 补码运算题 | 给定位数，完成加减法。 | 写补码、列加法、判断溢出、写真值结果。 |
| 浮点运算题 | 浮点加减或规格化。 | 按对阶、运算、规格化、舍入写步骤。 |
| ALU 结构题 | 说明 ALU 功能或控制。 | 从算术、逻辑、移位、标志位四点回答。 |

## 通用答题模板

1. 判断题目涉及的知识点；
2. 写出教材中的相关概念、规则或公式；
3. 按步骤展开分析或计算；
4. 给出结论；
5. 标注需要人工复核的 OCR、公式或图表内容。
