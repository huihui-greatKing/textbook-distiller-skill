# 第2章 计算机中的数据表示 题型与作业方法

| 题型 | 常见考法 | 解题/作业写法 |
| --- | --- | --- |
| 数制转换题 | 二、八、十、十六进制之间转换。 | 写出转换依据，保留中间步骤，避免只给结果。 |
| 机器数表示题 | 把真值写成原码、补码、反码或浮点格式。 | 先判断位数和符号，再写编码过程。 |
| 编码比较题 | 比较原码、补码、反码、定点数和浮点数。 | 从表示方式、范围、运算方便性比较。 |
| 校验码题 | 奇偶校验或 CRC 相关题。 | 说明校验规则、生成过程和可发现的错误类型。 |

## 通用答题模板

1. 判断题目涉及的知识点；
2. 写出教材中的相关概念、规则或公式；
3. 按步骤展开分析或计算；
4. 给出结论；
5. 标注需要人工复核的 OCR、公式或图表内容。
