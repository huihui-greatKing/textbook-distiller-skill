# 第8章 输入/输出系统 题型与作业方法

| 题型 | 常见考法 | 解题/作业写法 |
| --- | --- | --- |
| I/O 方式比较题 | 比较查询、中断、DMA。 | 列出优缺点和适用场景。 |
| 中断过程题 | 说明中断响应和服务流程。 | 按流程写，不要只说“打断 CPU”。 |
| DMA 分析题 | 说明 DMA 如何传输数据。 | 写清 CPU、DMA 控制器、主存和外设分工。 |

## 通用答题模板

1. 判断题目涉及的知识点；
2. 写出教材中的相关概念、规则或公式；
3. 按步骤展开分析或计算；
4. 给出结论；
5. 标注需要人工复核的 OCR、公式或图表内容。
