# 测试提示词

每章至少覆盖学习理解、复习提纲、作业解题和易错辨析，检查 Skill 是否真的能帮助学习，而不是只输出概要。

## 第1章 绪论

- 应调用：`skills/chapter-01-introduction/SKILL.md`

### 学习理解型测试
1. 请讲透“计算机系统”，必须包含是什么、为什么、怎么用、怎么考、易混点。
2. 请画出第1章知识地图，并说明各知识点之间的关系。
3. 请解释本章为什么要学习“建立计算机系统的整体框架，理解硬件、软件、指令、程序和性能指标之间的关系。”。

### 复习提纲型测试
1. 请生成第1章考前 20 分钟复习提纲。
2. 请列出第1章必背概念、必会方法和高频题型。
3. 请给出第1章 8 个自测问题并附答题思路。

### 作业解题型测试
1. 请围绕“五大部件分析法”生成一道作业题，并给出完整作业式答案。
2. 请用第1章 Skill 解一道比较题，要求用表格作答。
3. 请用第1章 Skill 解一道综合题，要求包含题目分析、过程、答案、检查。

### 易错辨析型测试
1. 请说明第1章最容易混淆的 3 个概念，并给纠正方法。
2. 请指出第1章作业中最容易漏写的步骤，并给改正模板。
