# 评价标准

| 维度 | 检查问题 | 合格标准 |
| --- | --- | --- |
| 学习理解 | 是否讲清是什么、为什么、怎么用、怎么考 | 不是一句定义，能帮助学生理解 |
| 知识地图 | 是否呈现概念之间的层级和联系 | 能看出本章结构 |
| 概念精讲 | 是否逐个解释核心概念并指出易混点 | 至少覆盖主要概念 |
| 公式规则 | 是否写明含义、条件、步骤和常见错误 | 可用于解题 |
| 图表整理 | 是否说明图表作用、关键元素和题目用法 | 不确定图表标注待复核 |
| 题型训练 | 是否覆盖概念、简答、计算、分析、比较、综合、实验/设计 | 每类有入口、步骤、模板、示例和答案 |
| 作业答案 | 是否输出题目分析、已知条件、过程、答案、易错点、检查 | 不能只写方法名 |
| 复习价值 | 是否有必背概念、必会方法、高频题型和自测 | 可用于考前复习 |
| 反概要化 | 是否避免停留在章节概要 | 概要型输出判定不合格 |
| 证据与复核 | 是否标注 OCR、公式、图表、例题不确定项 | 不确定内容不写成确定结论 |
