# 教材蒸馏模板

本文件保存可复用模板。需要填写表格、生成章节 Skill 或规范输出时读取。

## 1. 输入检查表

| 检查项 | 是否满足 | 说明 |
| --- | --- | --- |
| 教材名称 | 是/否 |  |
| 教材文件 | 是/否 |  |
| 蒸馏范围 | 是/否 |  |
| 输出目标 | 是/否 |  |
| OCR 需求 | 是/否 |  |
| 图表公式保留 | 是/否 |  |
| 作业模板需求 | 是/否 |  |
| 课程报告需求 | 是/否 |  |
| 待人工复核项 | 有/无 |  |

## 2. 阶段输出格式

```markdown
### 本轮完成内容

### 新增或修改文件

### 关键结论

### 待人工复核项

### 下一步建议

### 是否继续
```
