| 蒸馏范围 | 是/否 |  |
| 输出目标 | 是/否 |  |
| OCR 需求 | 是/否 |  |

## 4. 章节 Skill 标准模板

```markdown
# 第X章 XXX Skill

## 1. Skill 目标

说明本 Skill 用于帮助学生学习本章、复习考试重点、掌握题型并完成作业式答案。不能只写本章概要。

## 2. 本章学习目标

说明学完本章应该会解释什么、会计算什么、会比较什么、会写什么作业答案。

## 3. 本章知识地图

用层级结构列出本章知识点之间的关系。

## 4. 核心概念精讲

每个概念按照以下结构写：
- 是什么；
- 为什么引入；
- 怎么用；
- 怎么考；
- 易混点。

## 5. 重要公式、规则与方法

每个公式或规则按照以下结构写：
- 公式/规则内容；
- 符号解释；
- 使用条件；
- 解题步骤；
- 常见错误。

## 6. 教材图表整理

每个图表按照以下结构写：
- 图表作用；
- 关键元素；
- 对应知识点；
- 题目中如何使用；
- 待人工复核项。

## 7. 典型题型与解题模板

每类题型按照以下结构写：
- 题型特征；
- 解题入口；
- 解题步骤；
- 作业书写模板；
- 示例题；
- 参考答案；
- 易错点。

## 8. 教材式表达模板

整理本章答题时常用的概念解释、比较分析、计算过程和综合分析表达。

## 9. 作业辅助模板

包括简答题模板、计算题模板、比较题模板、综合题模板。

## 10. 复习提纲

包括必背概念、必会方法、高频题型、易混点、自测问题。

## 11. 自测题与参考答案

至少包含 5 道基础题、5 道典型题、3 道综合题，并给出参考答案或答题思路。

## 12. 常见错误与纠正

列出常见错误，并说明如何纠正。

## 13. 测试提示词

覆盖学习理解、复习提纲、作业解题和易错辨析。

## 14. 待人工复核项

- 页码或页序：
- OCR 不清楚内容：
- 公式：
- 表格：
- 图示：
- 例题：

| --- | --- | --- | --- | --- |
|  |  |  | 纳入/合并/淘汰/待复核 |  |
```
