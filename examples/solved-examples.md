# 例题处理方式

本文件不直接照抄教材例题原文，而是说明后续整理例题时的统一格式。

## 例题记录格式

| 字段 | 说明 |
| --- | --- |
| 章节 | 例题所属章节 |
| 题型 | 概念题、比较题、计算题、过程分析题或综合题 |
| 教材依据 | 教材页码、PDF 页序或 OCR 文本包 |
| 题目要点 | 用自己的话概括题目要求 |
| 解题思路 | 说明应调用的概念、公式、规则或图表 |
| 关键步骤 | 保留必要计算或分析过程 |
| 待复核项 | OCR 不清楚的公式、图表、表格或题干 |

## 示例格式

```text
章节：第 X 章
题型：……
教材依据：……
题目要点：……
解题思路：……
关键步骤：
1. ……
2. ……
结论：……
待人工复核：……
```
